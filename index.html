Why Python?

  • Universal programming language
  • Readable source code
  • Highly extendable via modules
  • Batteries included for common tasks
  • Large community
  • Free software
  • Can integrate C/C++ or Fortran code (via Cython)

History of Python

  • 1989: developed by Guido van Rossum as a hobby project on Christmas
  • 1994: Python 1.0
  • 2000: Python 2.0
  • 2008: Python 3.0, backwards-incompatible release
  • 2010: Python 2.7, last release of Python 2
  • 2015: Python 3.5, matrix multiplication operator @
  • ...

Lambda functions

def filter(numbers, test):
    return [x for x in numbers if test(x)]

print(filter([1, 23, 42], lambda x: x < 40)) # [1, 23]

Python basics

Classes

# class definition
class LinearSystem:
    # constructor
    def __init__(self, A, b):
        self.A = A
        self.b = b

    # class method
    def getResidual(self, x):
        return self.b - self.A @ x
# instanciate class
ls = LinearSystem(A, b)

# run method on instance
ls.getResidual(x)

class LinearSystem:
    # constructor
    def __init__(self, A, b):
        self.A = A
        self.b = b

    # class method
    def getResidual(self, x):
        return self.b - self.A @ x
# solve.py
from numpy.random import rand
from numpy.linalg import solve

from utils import LinearSystem

ls = LinearSystem(rand(10, 10),
                  rand(10, 1))
x = solve(ls.A, ls.b)
print(ls.getResidual(x))
